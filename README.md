### [NetMaster — симулятор сетевого инжиниринга](https://github.com/gaspacho2103/netmaster)

**JavaScript, PHP, HTML/CSS, leader-line (визуализация связей)**

Интерактивный инструмент для проектирования и симуляции компьютерных сетей:

- Визуальное создание топологии сети (маршрутизаторы, коммутаторы, ПК)
- Настройка IP-адресов, подсетей, VLAN
- Симуляция передачи пакетов с отображением маршрута (leader-line)

---

### [АвтоТех — комплексная экосистема для автосервиса](https://github.com/gaspacho2103/autotechnical)

**PHP + REST API + Electron + MySQL**

Проект, объединяющий три компонента в единую систему управления СТО:

- **Клиентский веб-сайт** — запись на сервис, каталог услуг, отзывы.
- **Админ-приложение на Electron** — десктоп-клиент для сотрудников: управление заказами и клиентами.
- **Единое REST API на PHP** — обеспечивает связь между сайтом, админкой и базой данных.
